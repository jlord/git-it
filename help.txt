  GITコマンド
  ------------{/cyan}

  {cyan}おめでとうございます{/cyan}

  {bold}ユーザ名を設定{/bold}

  $ git config --global user.name "Your Name"

  {bold}ユーザ名を設定{/bold}

  $ git config --global user.username githubusername

  {bold}メールアドレスを設定{/bold}

  $ git config --global user.email youremail@example.com

  {cyan}チェックイン関連{/cyan}

  {bold}リポジトリの状況を確認する{/bold}

  $ git status

  {bold}変更の差分を表示する{/bold}

  $ git diff

  {bold}ファイルをコミットに追加する{/bold}

  $ git add <FILENAME>

  {bold}すべてのファイルをコミットに追加する{/bold}

  $ git add .

  {bold}変更をコメントを付けてコミット（保存）する{/bold}

  $ git commit -m "your commit message"

  {cyan}リモート関連{/cyan}

  {bold}リモートにPushする{/bold}

  $ git push <REMOTENAME> <BRANCHNAME>

  {bold}リモートから変更をPullする{/bold}

  $ git pull <REMOTENAME> <BRANCHNAME>

  {bold}リモートにURLを設定する{/bold}

  $ git remote set-url <REMOTENAME> <BRANCHNAME>

  {bold}リモートを追加(Add)する{/bold}

  $ git remote add <REMOTENAME> <BRANCHNAME>

  {cyan}ブランチ関連{/cyan}

  {bold}ブランチを、現在使っているブランチにマージする{/bold}

  $ git merge <BRANCHNAME>

  {bold}ブランチを変更する{/bold}

  $ git checkout <BRANCHNAME>

  {bold}ローカルのブランチを削除する{/bold}

  $ git branch -D <BRANCHNAME>

  {bold}リモートのブランチを削除する{/bold}

  $ git push <REMOTENAME> --delete <BRANCHNAME>

  {cyan}
  BASHコマンド
  -------------{/cyan}

  {bold}新しいフォルダ（ディレクトリ）を作成する{/bold}

  $ mkdir <FOLDERNAME>

  {bold}ディレクトリを移動する(Change Directory){/bold}

  $ cd <FOLDERNAME>

  {bold}一つ前のディレクトリに移動する{/bold}

  $ cd ..

  {bold}ホームディレクトリに移動する{/bold}

  $ cd

  {bold}フォルダ内のファイルのリストを表示する{/bold}

  $ ls

  {bold}ファイルを削除する{/bold}

  $ rm <FILENAME>

  {bold}フォルダを削除する{/bold}

  $ rm -rf <FOLDERNAME>
