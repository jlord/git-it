關卡內容：

在你的 repository 中建立一個新檔案，加入一些內容，並將那些修改 commit 到 Git。

步驟：建立一個新檔案

現在你有了一個 repository 了，新增一個檔案到裡頭吧！

打開一個文字編輯器。現在開始寫幾行文字，說聲嗨之類的，並將檔案儲存為 readme.txt 到你在上一關中所建立的 'hello-world' 資料夾。

步驟：status、add 及 commit 修改

接著檢查你的 repository 的 status，看看裡面是否有任何修改。回到你的終端機，應該還在你剛剛所建立的 'hello-world' 資料夾裡頭，直接用這個指令查看狀態：

$ git status

接著將剛剛建立檔案，add 進你想 commit（或儲存）的修改。

$ git add readme.txt

最後，將那些修改 commit 到 repository 的歷史紀錄當中，並隨附上一個簡短的異動說明。

$ git commit -m "<your commit message>"

步驟：進行更多修改

現在在多寫一行文字到 readme.txt 之中並存檔。

在終端機裡，你可以查看現在的檔案跟上次你所 commit 的檔案之間有那些 diff 存在。

$ git diff

利用你剛剛所學到的知識，commit 這個最新的修改。

確認你完成了所有步驟：

git-it verify

前往下一關：

git-it

撇步

  • 檢查 repository 中的修改 status
  • $ git status
  • 查看對檔案的修改
  • $ git diff
  • 準備 commit 對於一個檔案的修改
  • $ git add <FILENAME>
  • 準備 commit 對於所有檔案的修改
  • $ git add .
  • commit（或儲存）你的修改並附上簡短的異動說明
  • $ git commit -m "<your commit message>"
