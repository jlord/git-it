告訴 Git 你要指定哪個資料夾，然後 Git 就會開始記錄這資料夾裡的一切更動。不管是新增還是刪除檔案，甚至只是改變檔案中的一個字，Git 都會全部追蹤並加上時間標記。這就是版本控制。

使用終端機

終端機（或是 Bash）是一種只利用鍵盤輸入指令來操控電腦的方式。你可以重新命名檔案、開啟檔案、建立新資料夾、在不同目錄（資料夾）之間切換，全都利用鍵盤輸入指令完成。你甚至不需離開終端機，便可以在終端機裡運行文字編輯器（例如 Vim）。

除了瀏覽你的電腦，你也可以在終端機裡使用具有終端機界面（command-line interface，CLI）的程式，這類的程式可藉由輸入鍵盤指令來運行。Git-it 便是其中之一，你現在正透過終端機使用 Git-it！而 Git 是另一個例子。你可以在終端機裡用指令來存取與操控 Git，很快你就會學會怎麼做了！

在 Git-it 裡你將學會一些基本操作的指令，每個步驟中會解釋這些指令的功用。

步驟：建立一個新的 repository

你將新建一個資料夾，並把它設定成一個 Git repository。

為了更簡單完成這件事，把資料夾和專案取同樣的名字。就叫 'hello-world' 好了。

你可以將下列指令一一輸入至終端機。

建立新資料夾：

$ mkdir hello-world

進入這個資料夾：

$ cd hello-world

把這個資料夾設定成一個 Git 專案資料夾

$ git init

就是這樣！終端機會回應並切換到下一行。如果你想要確認資料夾是否已經是個 Git repository，輸入 git status，只要不是顯示 'fatal: Not a git repository...'，那就對了！

確認你完成了所有步驟：

git-it verify

前往下一關：

git-it

撇步

  • 建立一個新資料夾（或稱做目錄）
    • $ mkdir <FOLDERNAME>
  • 進入一個資料夾（或稱做切換目錄）
    • $ cd <FOLDERNAME>
  • 列出資料夾內容
    • $ ls
  • 為一個資料夾加上 Git 功能
    • $ git init
