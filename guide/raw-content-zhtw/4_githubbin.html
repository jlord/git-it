關卡內容：

建立一個 GitHub 帳號，並在 Git 設定中加入使用者名稱。

更好的協作

目前為止你建立的 repository 只存在你自己的電腦上，雖然方便，但若想要與別人分享或協作時就顯得困難許多了。別擔心，這就是為什麼我們需要 GitHub！

步驟：建立一個 GitHub 帳號

GitHub 是一個讓大家不論在哪裏都可以上傳他們的 Git 專案，並輕鬆協作的網站。

前往 github.com 並註冊一個免費的帳號。 水啦！歡迎歡迎！

步驟：告訴 Git 你的 GitHub 名稱

告訴 Git 你的 GitHub 名稱，這個 GitHub 名稱將會在之後的關卡中用到。注意輸入時必須與你在 GitHub 上建立的完全一致－大小寫必須一致。

告訴 Git 你的 GitHub 名稱：

$ git config --global user.username gitHubUsername

確認你完成了所有的步驟：

git-it verify

前往下一關：

git-it
