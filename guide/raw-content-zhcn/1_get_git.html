关卡内容：

请安装 Git 到你的电脑上，并且设定好 Git 内部的用户名和电子邮箱。

Git

Git 是一个开源软件（完全免费），作者为 Linus Torvalds，他也是 Linux 操作系统的作者。

Git 是一个专门做修改记录的程序，在软件、程序设计的领域中叫做版本控制。

步骤：安装 Git

  • Windows 系统：建议你下载 GitHub for Windows 这个软件。这个程序内包含 Git 在里面，而且比较容易安装： windows.github.com。你可以用 Git Shell 作為你的終端機。
  • Mac 系统: 你可以下载 GitHub for Mac 这个软件，里面也包含 Git 的程序。 mac.github.com（必须到「Preferences」中选择「Install Command Line Tools」）。或是：
    • 直接下载、安装 Git 主程序：前往 git-scm.com/downloads 并安装网页上的步骤安装。

Git 不像是你电脑里的其他软件。你不会在桌面上看到一个软件图标，但你可以通过终端或是其他 Git 的电脑程序（如 GitHub for Mac 或 GitHub for Windows）来使用。而终端就是我们在 Git-it 中将会使用的程序！

步骤：设置 Git

安装好 Git 后，打开 终端（或 Bash、Shell、Prompt、命令提示符），并输入以下命令来确认你有顺利安装好 Git：

$ git --version

如果有安装成功，这个命令将会告诉你，你所安装的版本号码，如：

git version 1.9.1

(请确定你的 git 版本是在 1.7.10以上。)

接下来，让 Git 知道这台电脑所做的修改该连接到什么使用者：

设置你的名字：

$ git config --global user.name "<Your Name>"

设置你的电子邮箱：

$ git config --global user.email <youremail@example.com>

确认你完成了所有步骤：

git-it verify

前往下一关：

git-it

提示

这个货币符号 $ 会常常出现在程序的文件中，这个符号代表这个指令是终端的指令（参考上面的命令写法）。在输入命令的时候是不需要输入 $ 的，只需要输入它后面的命令即可。例如如果你要输入最后面「确认完成步骤的命令」，只要在终端输入 git-it verify 就好了。
