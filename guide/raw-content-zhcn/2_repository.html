使用终端

终端（或是 Bash）是一种只利用键盘输入命令来操控电脑的方式。你可以重新命名文件、打开文件、建立新文件夹、并在不同目录（文件夹）之间切换，全都利用键盘输入命令完成。你甚至不需离开终端便可以在终端里运行文本编辑器（例如 Vim）。

除了浏览你的电脑，你也可以在终端里使用具有终端界面（command-line interface，CLI）的程序，这类的程序可通过输入键盘命令来运行。Git-it 便是其中之一，你现在正通过终端使用它！而 Git 也是另一个例子。你可以在终端里通过命令来存取与操控 Git，不久后你便会这么做了！

在 Git-it 里你将学会一些基本操作的指令，而在每个步骤中将会解释这些指令的作用。

步骤：建立一个新的 repository

你将建立一个新的文件夹，并把它设置成一个 Git repository。

为了让整件事容易一些，把你的文件夹取名为与项目同名。就叫 'hello-world' 好了。

你可以将下列指令一一输入至终端。

建立新文件夹：

$ mkdir hello-world

假如这个文件夹：

$ cd hello-world

把这个文件夹设置成一个 Git 项目文件夹

$ git init

就是这样！终端会回应并切换到下一行。如果你想要确认此文件夹是否已经是个 Git repository，输入 git status，只要不是显示 'fatal: Not a git repository...'，那就对了！

确认你完成了所有步骤：

git-it verify

前往下一关：

git-it

提示

  • 建立一个新文件夹 (或称作目录)
    • $ mkdir <FOLDERNAME>
  • 进入一个文件夹 (或称作切换目录)
    • $ cd <FOLDERNAME>
  • 列出文件夹内容
    • $ ls
  • 为一个文件夹加上 Git 功能
    • $ git init
