关卡內容：

建立一个 GitHub 账号，并在 Git 设定中加入使用者账号。

团结力量大

目前为止你建立的 repository 只存在你自己的电脑上，虽然方便，但若想要与別人分享或合作时就先得困难许多了。別担心，这就是为什么我们需要 GitHub！

步骤：建立一个 GitHub 账号

GitHub 是一个让大家不论在哪里都可以上传他们的 Git 项目，并轻松合作的网站。

前往 github.com 并注册一个免费的账号。 欢迎欢迎！

步骤：告诉 Git 你的 GitHub 账号

告诉 Git 你的 GitHub 账号，这个 GitHub 账号将会在之后的关卡中用到。注意输入时必须与你在 GitHub 上建立的完全一致──大小写必须一致。

告诉 Git 你的 GitHub 账号：

$ git config --global user.username <USerNamE>

确认你完成了所有步骤：

git-it verify

前往下一个关卡

git-it

git-it verify 的时候没有通过？

GitHub & Git config usernames do not match

常见的错误是 GitHub 账号跟 git config 设定的账号没有一致。譬如 'jlord' 跟 'JLord' 是不一样的哦。

要修改 Git 设定的账号，只要再用一次上面教的指令，大小写打对就可以了：

$ git config --global user.username <USerNamE>

更新完之后，再检查一次看看吧！
