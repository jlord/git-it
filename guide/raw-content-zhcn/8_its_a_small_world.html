关卡内容：

帮项目新增 collaborator。

写程序边交朋友

和别人一起在 GitHub 上合作写程序是最棒的事情了，因为 GitHub 让来自世界各地的人一起工作变得超简单。

Collaborators 是被赋予特定 repository 编辑权限的使用者。项目要新增 collaborators，先到 repository 的 GitHub 页面，按下右边侧栏的 'Settings'（板手和螺丝起子的图标）。选择 'Collaborators' 分页。输入账号，按 Enter 或是用鼠标按 'Add' 就完成了喔。

步骤： Hi！Reporobot！

去第六关 forked Patchwork repository 的 GitHub 页面，新增一位协作者：'reporobot'。网址看起来会像是下面这样，但 'YOURUSERNAME' 会换成你的账号哦。

github.com/YOURUSERNAME/patchwork/settings/collaboration

确认你完成了所有步骤：

git-it verify

前往下一个关卡：

git-it

提示

  • 从 remote pull 更新回来
    • $ git pull <REMOTENAME> <BRANCH>
  • 复制 repository 到自己的电脑里
    • $ git clone <URL>
  • 新增 remote connections
    • $ git remote add <REMOTENAME> <URL>
  • 查看现有的 remote connections
    • $ git remote -v
