关卡內容：

在你的 repository 中建立一个新文件，新增一些内容到那个文件并且将那些修改 commit 到 Git 中。

步骤：建立一个新文件

现在你有了一个 repository 了，新增一个文件到里头吧！

打开一个文本编辑器。现在开始写几行文字，说声Hi之类的，并将文件存储为 readme.txt 到你在上个挑战中所建立的 'hello-world' 文件夹。

步骤：status、add 及 commit 修改

接着检查你的 repository 的 status，看看是否在里头有任何修改。回到你的终端，应该还在你刚刚所建立的 'hello-world' 文件夹里头，直接用这个命令查看状态：

$ git status

接着将刚刚建立的文件，add 进你想 commit（或存储）的修改。

$ git add readme.txt

最后，将那些修改 commit 到 repository 的历史记录当中，并随附上一个简短的改动说明。

$ git commit -m "<your commit message>"

步骤：进行更多修改

现在再多写一行文字到 readme.txt 之中并存储。

在终端里你可以查看有哪些 diff 存在与现在的文件以及上次你所 commit 的文件之间。

$ git diff

利用你刚刚所学到的知识，commit 这个最新的修改。

确认你完成了所有步骤：

git-it verify

前往下一关：

git-it

提示

  • 检查 repository 中的修改 status
  • $ git status
  • 查看对文件的修改
  • $ git diff
  • 准备 commit 对于一个文件的修改
  • $ git add <FILENAME>
  • 准备 commit 对于所有文件的修改
  • $ git add .
  • commit(或存储）您所准备好的修改并附上一个简短的改动说明
  • $ git commit -m "<your commit message>"
